# Object-Oriented Programming in C++

This repository contains a comprehensive collection of Object-Oriented Programming (OOP) concepts in C++, ranging from basics to advanced topics. It is designed to help with learning C++ OOP and preparing for technical interviews.

## How to Run

You can compile the C++ files using a C++ compiler like `g++`.

Example:

```bash
g++ 01_Basics/class_object.cpp -o class_object
./class_object
```

## Prerequisites

- Basic understanding of C syntax (variables, loops, functions).
- A C++ compiler (GCC, Clang, MSVC).
